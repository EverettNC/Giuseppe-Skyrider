## 🌟 Core Philosophy

### Rules to Code By

1. **Nothing Vital Lives Below Root**
   - All core logic at root level - radical transparency in architecture
   - No hidden complexity, no buried dependencies
   - Anyone can fork, understand, and build on our work

2. **Proximity Principle**
   - Files that work together stay together
   - Function over abstraction
   - Cognitive load reduction for neurodivergent developers

3. **Style Unity**
   - Code reads like poetry
   - Clear, intentional, inviting
   - Accessibility in code structure itself

### Why This Matters

Traditional software development marginalizes neurodivergent thinkers through:
- Unnecessary complexity
- Gatekeeping through abstraction
- Cognitive overhead in navigating nested structures

We're proving there's a better way. And we're building it publicly, with Claude.

## 🛠️ Flagship Project: Giovanni Skyrider

**What it is:** A digital executive assistant with personality, built specifically for neurodivergent content creators.

**Why it matters:** Existing productivity tools assume neurotypical executive function. We needed something different.

### Technical Architecture

**Built with:**
- React 18 + TypeScript (type safety + flexibility)
- Framer Motion (accessibility-first animations)
- Zustand (predictable state management)
- Web Speech API (voice integration)
- Custom personality engine (mood-based AI responses)

**17 TypeScript files, 2,844 lines of code, 100% root-level organization**

### Core Features

#### 1. Dynamic Avatar Presence
- Not a chatbot. A **companion**.
- 5 mood states: swagger, motivational, sassy, caring, hype
- Persistent on-screen presence
- Voice synthesis with personality

#### 2. Smart Medication Reminders
- Time-based alerts with adjustable "sass levels"
- Context-aware messaging
- Never patronizing, always motivating
- Example: *"Time to take those sexy little memory pills, babe."*

#### 3. Content Scheduler
- Visual calendar for content creation
- Medication + wellness + creative tasks
- Real-time monitoring
- Proactive notifications

#### 4. Photo Curator
- Organization by **vibe** (not folders):
  - 💅 Powerful Bitch Energy
  - 🌊 Quiet Storm
  - ✨ Sensual Motivational
  - 🎨 Creative Chaos
  - 🧠 Neurodivergent Pride
  - 🎬 Behind The Scenes
- Engagement score predictions
- "What Should I Post?" AI recommendations

#### 5. Social Media Manager
- Platform-specific caption generation (TikTok, Clapper, Facebook, Instagram)
- Hashtag strategy based on content vibe
- Best posting time recommendations
- Draft management system

#### 6. Studio Organizer
- Visual workspace zones
- Item tracking by category
- Real-time organization status
- Giovanni's decluttering advice
- Philosophy: *"Organization isn't about perfection - it's about knowing where your chaos lives."*
